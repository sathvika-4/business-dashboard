# E-commerce Business Dashboard

## Overview
An interactive Streamlit dashboard built using the Kaggle E-commerce dataset.

## Features
- Total Revenue KPI
- Total Orders KPI
- Total Customers KPI
- Quantity Sold KPI
- Country Filter
- Year Filter
- Month Filter
- Revenue by Country Treemap
- Top 10 Products
- Monthly Revenue Trend
- Download Filtered Data
- Business Recommendations

## Technologies
- Python
- Streamlit
- Pandas
- Plotly

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
